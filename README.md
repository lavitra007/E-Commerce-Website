# Farout Luxuries - Premium E-Commerce Platform

A luxurious, high-end E-Commerce website inspired by premium brands (Chanel, Dior, Jimmy Choo). Features a minimalist Black, White, and Dusty Pink color palette designed to seamlessly elevate the online shopping experience.

## ✨ Features
- **Luxury Aesthetics**: Custom Vanilla CSS design system guaranteeing a pristine, expensive feel.
- **Dynamic Routing**: Built for instantaneous loading via React Router. Includes customized views for (`/`, `/login`, `/register`, `/wishlist`, `/product/:id`, `/cart`).
- **State Synchronization**: Automatically checks backend JWT authorization and cleanly drops sessions on the frontend if database irregularities are detected.

## 🚀 Tech Stack
- Frontend: **React 19 + Vite**
- Routing: **React Router DOM**
- Styling: Custom **CSS Variables** defining brand identity parameters.
- API Connectivity: **Axios** 

## 🛠 Setup Instructions
1. Enter the client directory: `cd client`
2. Install dependencies: `npm install`
3. Start the Vite development server: `npm run dev`
